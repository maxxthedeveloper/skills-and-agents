# Vendor Billing Reference

**IMPORTANT**: Always web search first to find current invoice location. This list is a fallback only — billing portals change frequently.

## Typical Vendor Categories

### Dev Tools (often Stripe-based)
- Cursor - usage-based billing, Stripe portal
- Vercel - vercel.com/account/billing
- Supabase - often billed via Vercel integration
- Ngrok - dashboard.ngrok.com
- Convex - Stripe portal

### Design Tools
- Figma - figma.com/settings (billing section)
- Jitter - Stripe portal
- Fermat - Stripe portal

### Productivity
- Notion - notion.so/settings (plans & billing)
- Loom - loom.com/settings/billing
- MyMind - Stripe portal
- Wispr - Stripe portal

### Infrastructure
- Google Workspace - admin.google.com/ac/billing
- Claude.ai (Anthropic) - claude.ai/settings/billing

### Marketplaces
- Gumroad - gumroad.com/library (purchase receipts)

### Dutch Services
- Bunq - bunq app or web portal (facturen section)
- Tellow - tellow.nl/instellingen/facturen

## Invoice Search Patterns

When web searching, try these query patterns:
1. `"[vendor]" download invoice`
2. `"[vendor]" billing portal invoice PDF`
3. `"[vendor]" receipt download`
4. `site:[vendor domain] invoice OR billing OR receipt`

## Stripe-Invoiced Vendors

Many SaaS tools use Stripe for billing. If a vendor uses Stripe:
1. Check email for Stripe receipt links
2. Or: billing.stripe.com (if you have the Stripe customer portal link)
3. Login via email magic link or Google OAuth
